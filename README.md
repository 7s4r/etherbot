# Etherbot		
 		
## Setup		

### Dev requirements		

In order to run the project in local you will have to Docker >= 1.12 and docker/compose >= 1.8		

## Start		
In the project root directory run the following command:		
```		
docker-compose up -d		
```		

## App		

Install dependencies :		
```		
docker-compose exec app npm install		
```
